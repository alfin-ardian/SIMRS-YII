# Bagaimana Cara Instalasi

1. Clone project ini
2. cd `SIMRS-YII`
3. Ketik `composer install` atau `composer update`
4. Buat database pada posgresql
5. Jalankan migrasi dengan perintah `yii migrate`
6. Jalankan aplikasi dengan `php yii serve`

# Thats it!! Enjoy!
